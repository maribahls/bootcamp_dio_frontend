## Projeto Contador
Um contador simples implementado em HTML, CSS e JavaScript.

## Como Usar

1. Abra o arquivo `index.html` em um navegador da web.
2. Clique no botão "+" para incrementar o contador.
3. Clique no botão "-" para decrementar o contador.
